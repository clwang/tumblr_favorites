# Tumblr Favoriting app

A web app that allows you to search tumblr blogs by name and tag. Favorite posts that you like!

## Usage

Clone/Fork the project. 

```ruby
bundle install
rails server
```


## Tech stack

* Ruby on Rails
* React
* Bootstrap for styling
